# Retrieval-Augmented-LLM-Based-PDF-Query-Engine
An AI-powered document assistant that allows users to upload a PDF and ask questions about its content using a Large Language Model (LLM).
It is an intelligent PDF Question Answering system built using a Retrieval-Augmented Generation (RAG) pipeline.  

 Built using Python, Gradio, and Groq API.

---

##   Features

-   Extracts text from PDF documents
-   Splits document into semantic chunks
-   Generates embeddings using SentenceTransformers
-   Performs similarity search using cosine similarity
-   Generates contextual answers using Google Gemini LLM
-   Efficient embedding caching for faster re-runs
